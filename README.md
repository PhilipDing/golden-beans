# 🌟 金豆豆存钱罐 🌟

一个简单易用的个人理财存钱应用，帮助养成储蓄习惯，让每一分钱都产生价值。

## ✨ 功能特性

- 💰 **存入/支出管理** - 轻松记录每一笔存入和支出
- 📈 **自动计息** - 年利率 6%，按月复利计算利息
- 📋 **存取记录** - 完整的交易历史记录
- 🧮 **利息计算器** - 预览不同存入金额和时间的收益
- ☁️ **云端存储** - 数据存储在 jsonbin.io，多设备同步
- 📱 **响应式设计** - 完美适配手机、平板和电脑
- 🎨 **精美界面** - 现代化 UI 设计，操作流畅

## 🚀 快速开始

### 前置要求

- 现代浏览器（Chrome、Firefox、Safari、Edge 等）
- jsonbin.io 账号和 API Key

### 安装

1. 克隆或下载项目
```bash
git clone <repository-url>
cd golden-beans
```

2. 配置 jsonbin.io

在 [jsonbin.io](https://jsonbin.io/) 注册账号并创建一个新的 bin，然后在 `api.js` 中配置你的 API Key 和 Bin ID：

```javascript
const jsonBinAPI = new JsonBinAPI(
    'YOUR_API_KEY',
    'YOUR_BIN_ID'
);
```

3. 打开应用

直接在浏览器中打开 `index.html` 文件即可使用。

## 🎯 使用说明

### 存入金豆豆

1. 点击【存入】按钮
2. 输入存入金额，或选择快捷金额（10元、50元、100元、200元）
3. 点击【确认存入】
4. 等待数据保存完成

### 支出金豆豆

1. 点击【支出】按钮
2. 输入支出金额，或选择快捷金额
3. 点击【确认支出】
4. 等待数据保存完成

### 利息计算器

1. 点击【利息计算器】按钮
2. 输入存入金额和存入时间（月）
3. 点击【计算】按钮
4. 查看本金、利息和本息合计

## 💾 数据存储

应用使用 jsonbin.io 进行云端数据存储，数据结构如下：

```json
{
  "balance": 700.00,
  "records": [
    {
      "type": "deposit",
      "amount": 700,
      "date": "2026-01-01T00:00:00.000Z",
      "icon": "🪙",
      "label": "存入"
    }
  ],
  "lastInterestDate": "2026-01-01T00:00:00.000Z"
}
```

## 📊 利息计算

- **年利率**：6%
- **月利率**：0.5%
- **计息方式**：按月复利
- **计算公式**：`本息合计 = 本金 × (1 + 月利率)^月数`

## 🔧 开发说明

### 本地开发

1. 修改代码后直接在浏览器中刷新页面即可查看效果
2. 使用浏览器开发者工具调试
3. 控制台可查看 API 调用日志

### API 调用优化

- 首次加载时只调用一次 API 获取所有数据
- 使用 localStorage 记录初始化状态，避免不必要的 API 调用
- 存入/支出操作时显示 loading 状态，防止重复提交

## 📝 注意事项

- 首次使用会自动初始化 700 元初始金额
- 利息在每月 1 号自动计算并添加到余额
- 数据存储在云端，请妥善保管 jsonbin.io API Key
- 建议定期备份数据
